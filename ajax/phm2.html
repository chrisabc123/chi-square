fault detection and breakdown prediction

Project Info:

Most of the data may be unorganized, unlabeled, or just a freshly made system that cannot be trained. This is especially true in the field of PHM, where sensor data is used to manage machine plant systems. The model i present is particularly applicable to data with these characteristics. It would be nice to have data extracted from a specific failure state, but anomaly detection is performed using data extracted from the existing normal state, including when there is no.

This project extracts features that can be kept constant over time from time series data in a steady state, and proposes a new index to explain the condition of the system. In addition, the new index can be used to statistically determine a defect, and if there is training data, it can be diagnosed and predicted using a trend.

This model will be applicable to other fields as well.
